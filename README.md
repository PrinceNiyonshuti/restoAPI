## Restaurants API

Simple Restaurants API to be consumed by any frontend 

### after run these script 
    composer install
    cp .env.example .env or copy .env.example .env
    php artisan key:generate
    php artisan migrate
    php artisan db:seed
    php artisan storage:link
    php artisan queue:work
    php artisan serve
